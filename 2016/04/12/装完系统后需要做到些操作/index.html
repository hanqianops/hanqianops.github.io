装完系统后需要做的一些操作

  • 系统

操作系统选择优化
使用64位的系统，以便支持更大的内存等系统资源
可不配置SWAP，如果内存不足，增加更多内存
文件系统调优，给数据仓库一个单独的文件系统，一般效率更高、更可靠
给磁盘设置合适的block大小，减少IO读写次数
不要使用NFS。光纤存储，SAN架构


关闭不需要的系统服务

  • iptables、rsyslog、crond、sshd、network、irqbalance (充分利用CPU的多核，提升性能） haldaemon (鼠标键盘等） messagebus（图形界面需要） sysstat
  • chkconfig –list|awk ‘{print $1}’|xargs -I {} chkconfig {} off
  • chkconfig –level 3 crond on
  • chkconfig –level 3 rsyslog on
  • chkconfig –level 3 network on
  • chkconfig –level 3 sshd on


关闭SElinux

vim/etc/selinux/config
SELINUX=enforcing改为disabled

setenforce 0 临时关闭selinux
sestatus -v 查看selinux状态


关闭IPv6

/etc/modprobe.d/dist.conf 增加：
alias net-pf-10 off
options ipv6 disable=1


##　关闭tty、禁止键盘重启(/etc/inittab)
/etc/init/start-ttys.conf /dev/tty[1-6] ==> /dev/tty[1-3]
/etc/sysconfig/init ACTIVE_CONSOLES=/dev/tty[1-6] ==>ACTIVE_CONSOLES=/dev/tty[1-3]
initctl stop tty TTY=/dev/tty6 #临时禁止tty
删除特殊的用户和用户组
因为账户越多，系统就越不安全从而很可能被黑客利用，威胁服务器的安全。
可删除的用户:adm、lp、sync、shutdown、halt、news、uucp、operator、games、gopher等。
可删除的用户组:adm、lp、news、uucp、games、dip、pppusers、popusers、slipusers等。

SSH优化

使用公钥登录
改变端口号
禁止空密码登录
禁止root远程登录
优化SSH
GSSAPIAuthentication no
UseDNS no


挂载noatime，被访问时不会产生写磁盘的I/O了

/etc/fstab修改:
/dev/sda1 / ext4 defaults,noatime 1 1
mount -o remount /


chattr锁定系统关键文件

chattr +i 文件路径 -i 解锁
a ：文件将只能增加数据，而不能删除
c ：这个属性设定之后，将会自动的将此文件『压缩』，在读取的时候将会自动解压缩，但是在储存的时候，将会先进行压缩后再储存(看来对于大文件似乎蛮有用的)
i ：不能被删除、移动、修改，设定连结也无法写入或新增资料对于系统安全性有相当大的帮助！
如：/etc/passwd、/etc/shadow、/etc/group、/etc/gshadow、//etc/services、/etc/inittab、/etc/rc.local
lsattr 查看文件状态


增加文件描述符

vim/etc/security/limit.conf

* soft nofile 60000
* hard nofile 65535

或者echo ulimit -SHn 65535 >> /etc/init.d/rc.local


配置sudo对普通用户权限精细控制

visudo 或 vim /etc/sudoers


grub加密

防止他人修改grub。用单用户模式启动
在title上插入password –MD5 密钥 #限制进入单用户模式
在title下插入password –MD5 密钥 #加密系统引导
grub-MD5-crypt 获得加密密钥


history

监控/var/log/secure,ssh的用户登录和注册日志
减少命令历史条目(/etc/profile)
HISTFILESIZE=100 .bash_history文件中最多只有100行
HISTSIZE=50 history 命令输出的记录数
HISTTIMEFORMAT=”%Y%m%d-%H%M%S: “
export HISTTIMEFORMAT

退出时自动清空history
echo -e “history -c \n clear” >>~/.bash_;ogout

终端限制600秒自动注册
echo “export TMOUT=600” >>~/.bash_profile

不记录敏感历史命令
HISTCONTROL=ignorespace 执行命令时要以空格开头


cron或NTP配置时间同步

*/5 ntpdate ntp.api.bz
