ps

  • 系统性能

linux上进程有5种状态:

  1. 运行(正在运行或在运行队列中等待)
  2. 中断(休眠中, 受阻, 在等待某个条件的形成或接受到信号)
  3. 不可中断(收到信号不唤醒和不可运行, 进程必须等待直到有中断发生)
  4. 僵死(进程已终止, 但进程描述符存在, 直到父进程调用wait4()系统调用后释放)
  5. 停止(进程收到SIGSTOP, SIGSTP, SIGTIN, SIGTOU信号后停止运行运行)

ps命令标识进程的5种状态码:

  1. D 不可中断 uninterruptible sleep (usually IO)
  2. R 运行 runnable (on run queue)
  3. S 中断 sleeping
  4. T 停止 traced or stopped
  5. Z 僵死 a defunct (”zombie”) process

命令参数

  • a 当前终端下的所有进程,包括其他用户
  • u 以用户为主输出进程信息
  • x 显示当前用户在所有终端的进程信息
  • -e 显示系统所有进程信息
  • -l 长格式显示进程信息
  • -f 使用完整的格式显示进程信息

以上是常用的几个命令参数

常用的命令参数组合

查看系统所有进程

  • ps -ef

查看所有线程

  • ps xH

查看某个进程的线程

  • ps -mp pid
  • 或者使用top命令： top -p pid -H

命令输出字段含义

  • VSZ: 占用虚拟内存
  • RSS： 占用常驻内存
  • TTY:在那个终端允许，？表示未知或不需要终端
  • START: 启用该进程的时间
  • TIME:占用CPU时间
  • PSR：表示占用的哪个CPU
  • STAT项：S 休眠 R运行 Z僵死(杀父进程) <高优先级 N低优先级 s父进程 +前台运行
